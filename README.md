# Coding With Mamta 🚀

A Gen-Z style, dark-theme cyberpunk website for the **Coding With Mamta** YouTube channel.

## 🌐 Live Preview
Open `index.html` directly in a browser, or serve locally:
```bash
npx serve .
```

## ✨ Features
- **Cyberpunk-meets-Glassmorphism** dark theme
- **Neon purple & cyan** accents with multi-layer glow effects
- **Anti-gravity float animations** — hero text and video cards float endlessly
- **3D card tilt** on hover (mouse-tracking perspective)
- **Particle system** background (45 animated particles)
- **Filter tabs** — All / Python / Web Dev / JS / ML
- **Glowing Download Resources buttons** per video card
- **Animated counters** — subscriber/view counts roll up on scroll
- **Download modal** with keyboard (Esc) support
- **Scrolling ticker** marquee of topics
- **Cursor glow trail** on desktop
- **Random glitch flicker** on hero title
- **Fully responsive** — Mobile / Tablet / Laptop / Desktop

## 📁 Project Structure
```
├── index.html       # Main HTML — semantic, SEO-optimised
├── style.css        # Cyberpunk CSS + all animations
├── script.js        # Interactive JavaScript
└── images/          # AI-generated cyberpunk thumbnails (6)
    ├── thumb1.png   # Python Basics
    ├── thumb2.png   # JavaScript Mastery
    ├── thumb3.png   # HTML & CSS
    ├── thumb4.png   # React JS
    ├── thumb5.png   # Data Structures & Algorithms
    └── thumb6.png   # Machine Learning
```

## 🎨 Tech Stack
- **HTML5** — Semantic markup, ARIA labels, SEO meta tags
- **Vanilla CSS** — Custom properties, glassmorphism, keyframe animations
- **Vanilla JavaScript** — IntersectionObserver, particle engine, 3D tilt

## 📱 Responsive Breakpoints
| Breakpoint | Layout |
|---|---|
| ≥ 960px | 2-col hero, 3-col video grid |
| 820–960px | Stacked hero, 2-col video grid |
| 640–820px | Hamburger nav, 1-col video grid |
| ≤ 640px | Compact mobile layout |
| ≤ 480px | Extra small — scales down all elements |
| ≤ 360px | Minimum phone support |
